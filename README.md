# Android

Bem-vindo ao repositório da aula de Desenvolvimento para Dispositvos Móveis II,  este repositório tem como função armazenar os projetos e exercícios desenvolvidos em aula.

## Sobre
Os projetos desenvolvidos tem como objetivo principal entender as funcionalidades de multiplas telas e o funcionamento de dados dinamicos, tanto dados resgatados do proprio celular, quanto vindos de um servidor externo (Firebase)
